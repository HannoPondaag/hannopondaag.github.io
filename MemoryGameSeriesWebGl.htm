WebGL memory spellen. Ook geschikt voor tablets (Android/IPad) & Smartphone ! Zonder netwerk mogelijkheden, play in a Mahjong-like style.

Java webstart versies worden niet meer ondersteund ! (Geverifieerd 10 september 2022)

  • Memory Game WebGL Versie Dieren: een Unity versie voor direct in een webpagina , simpel zonder Netwerk mogelijkheden, geprogrammeerd mbv Visual Studio Community 2017 & Unity.

  • Memory Game WebGL Versie Vogels : een Unity Memory Spel met Vogels voor het Web (simpel zonder Netwerk mogelijkheden), geprogrammeerd met Visual Studio Community 2017 & Unity.

  • Memory Game WebGL Versie Bloemen : een Unity Memory Spel met bloemen, versie voor het Web (simpel zonder te kunnen spelen met 2 spelers via het Net), geprogrammeerd met Visual Studio Community 2017 & Unity.

  • Memory Game WebGL Versie Alfabet : een Unity Memory Spel met het alfabet, versie voor het Web (simpel zonder Netwerk mogelijkheden), geprogrammeerd met Visual Studio Community 2017 & Unity.

  • Memory Game WebGL Versie Grieks Alfabet : een Unity Memory spel met het griekse alfabet, versie voor het Web (simpel zonder Netwerk mogelijkheden), geprogrammeerd met Visual Studio Community 2017 & Unity.

  • Memory Game WebGL Versie Getallen : een Unity Memory Spel met getallen, versie voor het Web (simpel zonder Netwerk mogelijkheden), geprogrammeerd met Visual Studio Community 2017 & Unity.
